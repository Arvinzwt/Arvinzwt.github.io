## CRM

```bash
# 第一次拉取请安装依赖
$ npm install

# 本地开发
$ npm run develop

# 文件打包
$ npm run build

# 在本地查看你的生产环境版本
$ npm run serve

# 清除缓存并重新开始
$ npm run clean
```
